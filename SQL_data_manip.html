SQL Database Management and Data Manipulation

There are multiple instances of code through this showcase. Each iteration of the code will be broken down and digested. To start I used SQL Server 2022 to create a database on my device. Secondly, I installed SQL Server Management Studio to access the database and manipulate the data. The dataset used is COVID-19 data obtained from https://ourworldindata.org/coronavirus. This dataset was then imported onto my database in two digestable tables, CovidDeaths and Covid Vaccinations.
